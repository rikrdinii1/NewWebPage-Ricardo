Conexión con Python a la API de Spotify

Proyecto de fin de modulo 1 del Bootcam de Data Analytics en Ironhack

Durante este modulo se ve el uso de Python y se termina con WebScraping y conexión a API´s para extracción de información

Mi proyecto consiste en la conexión a la API de Spotify para extraer información de mis gustos de música y los de mi pareja para poder analizarlos posteriormente.

Ya que la información viene en formato JSON, se trabaja con ella para poder dejarla en tablas de pandas y poder exportarla a un CSV y que sea mas manejable.

Por ultimo, hago mención que no se utiliza la ya creada Librería de Spotify, sino que me enfoco en la creación de toda la conexión a la API mediante la documentación oficial de Spotify.

Puedes revisar el repositorio de este proyecto en mi GitHub
